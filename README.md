# mk_framework

muzzik 开发的 cocos creator 游戏开发框架，目前处于开发阶段

路线图：https://trello.com/b/ILu6BXyX/trade-offs

---

## 构建框架声明文件(d.ts)

### 初始化

1. `cd tool\api-extractor`

2. `npm i`

### 运行

1. `npm run 构建框架声明文件`

## 构建框架文档

### 初始化

1. `cd tool\doc-website`

2. `npm i`

### 运行

1. `cd tool\api-extractor & npm run 构建框架文档`

2. `cd tool\doc-website & npm run start`
