## 🛠️ 安装框架

### 依赖环境
- NodeJs

### NPM 方式
1. 打开终端

1. 执行 `npm i -g @muzzik/mk-install`，等待安装完成
1. 执行 `npx mk-install`
1. 输入你的项目路径回车
1. 输入 `install` 回车，等待安装完成
